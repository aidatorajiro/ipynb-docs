フーリエ変換の基礎知識。¶

フランスの物理学者、ジョゼフ・フーリエ男爵(1768~1830)が、フーリエ変換という理論を18世紀の終わり頃に展開した！！ その理論は、なんと音や光といった自然界に存在する波動を、

$ y = 10sin(2x) + 5sin(9x) + 80sin(7.9x) $

みたいな感じに、数式に変換することができてしまう、という代物！！！、なんです。
数式になってしまえば、もうそれはデータの塊みたいなものですね〜。
だから、これがきちんとしたデータとして使える、と言ったんです！

ここに出てくる $ sin $ というのは、三角関数というものです。
三角関数をグラフにするとこんな形になります。 これは $y = sin(x)$ のグラフです。

次は、本格的にフーリエ変換、なるものが具体的にどんなものなのかを言いたいと思います。

What is "Fourier Transform"?¶

フーリエ変換というのは、数学の世界でもよく使われていますが（完全な数式の中で動かす場合はフーリエ級数展開と呼ばれる）、
工学・科学の世界でも大変重宝されています。とっても社会に役立っています。

まず、フーリエ変換と$sin$が生み出す波の、数学的な特性みたいなものを話します。

冒頭で、$ y = 10sin(2x) + 5sin(9x) + 80sin(7.9x) $という関数を見せました。（図１のグラフではないです。）

実際にフーリエ変換したときには、上の式のように数個の項で構成されているわけではなく、何千個、何万個の項で構成されています。
数学の世界だと項が「無限」に存在することになっています。カッコイイですね。（実際に現実世界でフーリエ変換するとなると、さすがに「無限」はないです）
そして、$sin$ が生み出す波にはとても興味深い特性があります。やばいです。
先ほどの $ y = 10sin(2x) + 5sin(9x) + 80sin(7.9x) $ のような関数は、

$a \times sin(bx)$

といった項の組み合わせで表現できます。aとbにそれぞれ10だとか2だとか、数字を入れて、項が出来上がります。こう言った項が「無限」に組み合わさって、波を表現します。
それで、例えば、ピアノで「ド」の音を鳴らすとします。しかし、それを録音したとしても、図１のような曲線にはなりません。
図１の曲線は一番単純な波だからです。ピアノが出す音は、複雑で、異なる「周波数」の音が組み合わさって出来たものなのです。
周波数というのは音で言えば「音の高さ」のことで、単位はHzです、というのは習いました。
上の式では $b$ の値が、その周波数です。$b$ の値を大きくすればするほど、高い音になります。
そして、$a$の値は振幅、音の大きさです。$a$ の値を大きくすればするほど、大きい音になります。

フーリエ変換では、波動のデータを、$a$という値と$b$という値に変換することができるできるのです！

フーリエ変換というのは、そういうものだからです。
フーリエ変換というのは、とても規則的な、数学的に生み出された、それこそ、この自分のコンピュータの中で生み出した、「図１」みたいな曲線を入れないと、きれいな答えは出してくれません。というか、規則的な波形を入れないときれいな答えどころか全く使い物になりません。
数学の世界では$sin$が作り出す波というのは永遠に同じパターンが繰り替えされますが、現実世界だと0.5秒間「ド」の音がなったら、今度は0.5秒間「レ」の音が鳴ったりして、永遠に同じ音、同じパターンというわけにはいきません。
ですが、そんな現実世界であっても、瞬きにも満たない、ほんの「一瞬」ですが（具体的には0.01秒ほど）、
その一瞬だけ、同じパターンの波が数回ほど繰り返されていたり、するのです。
（数回ほど、というのは少ない気もします。例えば「ド」の音だと、１秒間にたったの$260回$しか、振動を許されていないという・・・。ちょっと神様はケチですよね。）
やっぱり音を形作るのは波、それもパターンのある波ですので、それがなければ音程などは成り立ちません。だから、きちんとパターンはあります。
そのパターンが維持されるのはほんの一瞬の間だけです。が、ほんの一瞬でも、積もれば山となる。それを繰り返せばいいだけの話です。
「ほんの一瞬」だけ切り取って、フーリエ変換する。それを何百回と繰り返せば１秒になって、
その１秒の間、音の流れがどのように変化していったのかを知ることができます。

まあ、長い間同じ音を出せば、もちろん「ほんの一瞬」とは言わず、ずっとパターンは維持されますが・・・
この場合は「小は大を兼ねる」です。小刻みに分けて考えれば、どんな大きくて複雑なものも、飲み込める、というわけです。

このグラフの場合、横軸が「時間」で、縦軸が「周波数」、そして色は、もうみなさんお気づきかと思いますが、「音量」です。
今まで皆さんには、横軸「時間」がない状態、ほんの0.01秒間の間だけの解析を見せていました。
そして、今の0.01秒間が終わったら、次の0.01秒間に移って、ということを繰り返して行くと、だんだんと横軸「時間」が出来上がってきます。そうやって切ったかまぼこを並べるようにして、少しずつ積み上げていきます。
新たにできた次元、「音量」については、色が赤に近づけば近づくほど、その時間の、その周波数において、大きな音が出た、ということを意味します。かまぼこのピンク色の部分ですね。
逆に青色とかだと、ほぼその周波数には音がない、ということを意味します。
継続的に同じ周波数の音が続けば、継続的に、フーリエ変換した時の「突起」が同じ場所に現れるわけですから、赤い横線がその周波数の所に現れます。
よく見てみると、この画像にも、赤い横線が何本も引かれていますね。

# ======
#  STFT
# ======
def stft(x, win, step):
    inputlen = len(x) # 入力信号の長さ
    slicelen = len(win) # 窓幅、つまり切り出す幅
    flames = int(scipy.ceil(float(inputlen - slicelen + step) / step)) # スペクトログラムの時間フレーム数
    
    new_x = scipy.zeros(slicelen + ((flames - 1) * step), dtype = scipy.float64)
    new_x[:inputlen] = x # 信号をいい感じの長さにする
    
    X = scipy.zeros([flames, slicelen / 2], dtype = scipy.complex64) # スペクトログラムの初期化(複素数型)
    for i in range(flames):
        start = step * i
        F = scipy.fftpack.fft(new_x[start : start + slicelen] * win)
        X[i, :] = F[:len(F) / 2]
    
    return X

if __name__ == "__main__":

考察¶

ということで、実験の結果、「鼻歌にはフォルマントが現れない」ということがわかりました！
なぜこういう結論に至ったのかというと、フォルマントは、先ほど見せたように、もっと複数の周波数が合わさってできるものです。
このグラフはフォルマントと同じ細長い線が引かれてはいますが、フォルマントの場合はもっと線同士の密度が高いです。 これは、「鼻歌」というのは体の声を発する部分の中でも、割と単純な部分が振動しているからなのかもしれません。 あんまり厳密に実験していないから100%の説ではありませんが・・・。

考察¶

「あ」の音を鳴らす時と、「お」の音を鳴らす時では、似た部位、または同じ部位を一部共有して、音を鳴らしている、と考えました。
つまり、「あ」と「お」は音を鳴らす時に使う部位を一部共有している、という説です。
「あ」と「お」の音は母音同士のくせにお友達、というわけです。
これは「い」と「う」にも同じことが言えます。

まず、なぜそういう結論に至ったのかを説明します。 この仮説の実証方法はとても理科的でないので、理科的に言えば信ぴょう性はないかもしれませんが、でも信ぴょう性が「客観性」だというならば、格好自信があります。
まず、ミルフィーユみたいなのが時折曲がったりします。これはいうまでもなく、あ、い、う、え、お、の音の変わり目です。横軸が時間軸ですので、時間軸で見ると、ミルフィーユに五種類の変化があることがわかります。これは、まあ実証するまでもなく、あ、い、う、え、お、の五種類の音をだしたから、グラフ上に五種類のフォルマントができた、ということですね。
あ、い、う、え、お、をわかりやすく分けたものを用意しました。

で、よく見ると、「あ」のパターンと「お」がとても似通っていますよね。
また、「い」と「う」のパターンに至っては、ほぼ同じ、と言ってもいいでしょう。

そこで僕は、音を鳴らす時に使う部位を母音同士で一部共有しているのではないか、と考えました。
そして、問題はそれをどう実証するのかです。

僕は、実際に自分で声を上げて、その感覚で実証してみることにしました。とても原始的な方法ですね。
まず、「あ」と「お」の音を交互にだして、感覚をつかみました。
「あ」と「お」の音は、微妙に「お」の音の方が「あ」よりも下の方から出てきたのですが、それ以外はほぼ同じ感覚でした。非常に近い部位を共有している、という感覚です。これはグラフにも合致しています。グラフでも、「あ」と「お」は近いですが、ちょっと違っています。

次に、「い」と「う」のパターンでやってみました。
これもグラフどおりで、完全に同じ部位を共有しているな、という感覚でした。
念のため、「あ」と「い」、「あ」と「う」などのパターンもやってみましたが、ぜんぜん違う感覚でした。
でも、「い」と「う」になると、「い」は鼻を鳴らしていて、「う」は鳴らしていない、という違いしかなくて、その他の箇所は全くと言っていいほど、ありません。
